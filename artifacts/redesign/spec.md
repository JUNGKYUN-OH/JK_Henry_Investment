# spec: Henry Ledger — 무한매수법 전용 리디자인

## 개요

"라오어의 무한매수법"에 따라 ETF를 분할 매수·매도하는 사용자가 **매일 앱을 열어 탭 한 번으로 오늘의 매수를 완료**할 수 있는 시스템. 홈 화면이 "오늘 할 일"의 허브가 되며, 계획 없는 일반 거래는 제거하고 무한매수법 계획(Plan) 중심으로 앱 전체를 재편한다.

## 범위

### 포함

- **홈 화면 재편**: 오늘의 매수 할 일 목록, 진행도·목표가 요약, 포트폴리오 합산, 목표가 도달 배너
- **홈에서 바로 매수 실행**: 계획 항목 탭 → 현재가 자동 채움 → 수량 자동 계산 → 매수 완료
- **앱 진입 시 현재가 자동 조회**: 별도 버튼 없이 홈 진입 시마다 자동 갱신
- **계획 생성 시 파라미터 커스터마이징**: 분할 횟수(기본 40)·목표 수익률(기본 10%) 사용자 지정
- **거래일 기준 카운트**: 주말 및 미국 공휴일 제외, 실제 매수한 날만 진행 카운트
- **소급 매수**: 과거 거래일 날짜로 매수 기록 가능 (계획 상세에서)
- **매도 기록**: 보유 수량 범위 안에서 매도 거래 입력
- **포트폴리오 요약**: 전체 평가금액·수익률 (홈 화면)
- **티커 관리**: 추가·삭제 (계획 생성 전 사전 등록)
- USD 단일 통화

### 제외

- **계획 없는 일반 거래 (Buy/Sell without Plan)** — 무한매수법 전용 앱으로 전환; 모든 매수는 계획에 속함
- **라운드(회차) 자동 추적** — 매도 후 재시작은 사용자가 직접 새 계획 생성; 이력은 완료된 계획 목록으로 확인
- **건너뛰기 명시적 기록** — 오늘 매수 안 하면 그대로 둠; 별도 "건너뜀" 액션 없음
- **멀티유저·인증** — 현재는 단일 사용자; 향후 확장 대비 구조만 고려
- **차트·그래프 시각화** — MVP 이후
- **증권사 자동 연동** — 수동 입력 우선
- **세금 계산** — 별도 범위
- **국내 ETF** — 해외 ETF 먼저 검증

## 시나리오

### 1. 홈 진입 — 오늘의 할 일 표시 및 현재가 자동 조회

- **Given** 활성 계획이 1개 이상 있고, 오늘이 거래일(평일·미국 공휴일 아님)이며, 오늘 아직 매수 기록이 없는 상태
- **When** 홈 화면에 진입한다
- **Then** 현재가가 자동 조회되고 오늘 매수할 종목 목록이 표시된다

성공 기준:
- [ ] 홈 진입 시 별도 버튼 없이 Yahoo Finance에서 현재가를 자동 조회한다
- [ ] 조회 중에는 각 종목 행에 로딩 인디케이터가 표시된다
- [ ] 조회 완료 후 "오늘의 할 일" 섹션에 활성 계획의 종목명, 일 투자금액, 현재가, 예상 매수 수량(= 일 투자금 ÷ 현재가)이 표시된다
- [ ] 이미 오늘 매수 기록이 완료된 계획은 "오늘의 할 일"에 표시되지 않는다 (완료 표시로 구분)

### 2. 홈 진입 — 거래일이 아닌 날

- **Given** 오늘이 토요일·일요일 또는 미국 공휴일인 상태
- **When** 홈 화면에 진입한다
- **Then** 오늘은 거래일이 아님을 안내하고, 진행 중인 계획의 현황 요약은 표시된다

성공 기준:
- [ ] "오늘의 할 일" 섹션 대신 "오늘은 거래일이 아닙니다" 메시지가 표시된다
- [ ] 포트폴리오 요약(총 투자금, 평가금액, 수익률)과 계획별 진행도는 계속 표시된다

### 3. 현재가 자동 조회 실패

- **Given** Yahoo Finance API를 사용할 수 없는 상태
- **When** 홈 화면에 진입한다
- **Then** 기존 캐시 가격으로 표시되며 수동 입력 옵션이 제공된다

성공 기준:
- [ ] "현재가를 불러오지 못했습니다" 에러 메시지가 표시된다
- [ ] 이전에 조회된 캐시 가격이 있으면 해당 가격과 마지막 조회 시각이 함께 표시된다
- [ ] 각 종목에 수동 현재가 입력 필드가 노출되며, 입력 확인 후 예상 매수 수량이 재계산된다

### 4. 목표 매도가 도달 알림 배너

매도 기준은 **매수 진행 회차**에 따라 달라진다. 분할 횟수를 N이라 할 때:
- **N/2 회차 이하**: 현재가 ≥ 평균단가 × (1 + 목표 수익률) 시 전량 매도 신호
- **N/2 회차 초과**: 현재가 ≥ 평균단가 × 1.05 시 1차(50%) 매도 신호, 이후 현재가 ≥ 평균단가 × 목표 수익률 시 2차(나머지 50%) 매도 신호

*(기본값 N=40 기준: 20회차 이하 → 전량, 21회차 이상 → 분할)*

- **Given** 활성 계획이 있고 현재가가 매도 임계가 이상인 상태
- **When** 홈 화면에 진입한다 (현재가 자동 조회 후)
- **Then** 진행 회차에 따른 매도 신호 배너가 홈 최상단에 표시되고 [매도하기] 버튼이 제공된다

성공 기준:
- [ ] 진행 회차 ≤ N/2이고 현재가 ≥ 평균단가 × (1 + 목표 수익률)인 계획 → "[종목] 목표가 도달 (+X%) — 전량 매도" 배너 + [매도하기] 버튼이 표시된다
- [ ] 진행 회차 > N/2이고 현재가 ≥ 평균단가 × 1.05이며 1차 매도 미완료인 계획 → "[종목] 1차 매도 시점 (+5%) — 보유 수량의 50% 매도" 배너 + [매도하기] 버튼이 표시된다
- [ ] 진행 회차 > N/2이고 1차 매도 완료 후 현재가 ≥ 평균단가 × (1 + 목표 수익률)인 계획 → "[종목] 2차 매도 시점 (+X%) — 나머지 전량 매도" 배너 + [매도하기] 버튼이 표시된다
- [ ] 배너의 [매도하기] 버튼을 탭하면 해당 계획의 매도 확인 화면으로 이동한다
- [ ] 여러 종목이 동시에 매도 신호이면 각각 배너가 표시된다
- [ ] 어떤 매도 임계도 충족하지 않으면 배너가 표시되지 않는다

### 5. 홈에서 오늘 매수 실행

- **Given** 홈의 "오늘의 할 일"에 TQQQ 계획이 표시된 상태 (일 투자금 $100, 현재가 $50.00, 예상 수량 2.000)
- **When** TQQQ 항목을 탭하고 표시된 현재가·수량을 확인한 뒤 [매수 완료] 버튼을 탭한다
- **Then** 매수가 기록되고 계획 진행도와 목표 매도가가 즉시 갱신된다

성공 기준:
- [ ] 탭 후 매수 확인 화면에 종목명, 오늘 날짜, 현재가($50.00), 수량(2.000), 일 투자금($100.00)이 표시된다
- [ ] 현재가를 수정하면 수량이 (일 투자금 ÷ 새 단가)로 즉시 재계산된다
- [ ] 수량을 직접 수정하면 이후 단가 변경 시 수량이 자동 재계산되지 않는다
- [ ] [매수 완료] 탭 후 해당 계획의 진행도가 N+1일로 업데이트된다
- [ ] 계획 평균단가와 목표 매도가(= 계획 평균단가 × 목표 수익률)가 즉시 갱신된다
- [ ] 방금 매수한 TQQQ가 "오늘의 할 일" 목록에서 완료 처리된다
- [ ] 수수료 입력 필드가 있으며, 입력하지 않으면 $0으로 처리된다

### 6. 소급 매수 — 지난 거래일 매수 기록

- **Given** 어제 거래일에 TQQQ 매수를 못 한 상태 (계획 진행 중)
- **When** 계획 상세 화면에서 날짜를 어제로 선택하고 단가와 수량을 입력한 뒤 저장한다
- **Then** 해당 날짜의 매수가 기록되고 계획 카운트가 증가한다

성공 기준:
- [ ] 계획 상세에서 오늘 이전의 거래일 날짜를 선택하여 매수 기록이 가능하다
- [ ] 저장 후 계획 진행 카운트(완료 일수)가 1 증가한다
- [ ] 계획 평균단가와 목표 매도가가 즉시 재계산된다
- [ ] 이미 같은 날짜에 매수 기록이 있을 때 재입력 시도 → "해당 날짜에 이미 매수 기록이 있습니다" 경고가 표시되고, 사용자가 확인하면 저장된다
- [ ] 미래 날짜는 선택이 불가하다

### 7. 계획 생성 — 기본 파라미터

- **Given** TQQQ 티커가 등록된 상태
- **When** TQQQ, 총 투자금 $4,000으로 계획을 생성한다 (분할 횟수·목표 수익률은 기본값 유지)
- **Then** 분할 횟수 40, 목표 수익률 10% 기본값이 적용된 계획이 생성된다

성공 기준:
- [ ] 생성 후 계획 목록에 TQQQ 계획이 나타나고 진행 상황 "0 / 40일"이 표시된다
- [ ] 계획 상세에서 일 투자금 $100.00 (= $4,000 ÷ 40), 목표 수익률 10%가 표시된다
- [ ] 분할 횟수 미입력 시 기본값 40이 자동 적용된다
- [ ] 목표 수익률 미입력 시 기본값 10%가 자동 적용된다

### 8. 계획 생성 — 커스텀 파라미터

- **Given** SOXL 티커가 등록된 상태
- **When** SOXL, 총 투자금 $2,000, 분할 횟수 20, 목표 수익률 15%로 계획을 생성한다
- **Then** 입력한 파라미터가 적용된 계획이 생성된다

성공 기준:
- [ ] 생성 후 계획 목록에 진행 상황 "0 / 20일"이 표시된다
- [ ] 계획 상세에서 일 투자금 $100.00 (= $2,000 ÷ 20), 목표 수익률 15%가 표시된다
- [ ] 매수 기록 후 목표 매도가 = 계획 평균단가 × 1.15로 계산된다
- [ ] 분할 횟수에 1 미만 또는 소수 입력 시 에러 메시지가 표시된다
- [ ] 목표 수익률에 0 이하 입력 시 에러 메시지가 표시된다
- [ ] 총 투자금에 0 이하 입력 시 에러 메시지가 표시된다

### 9. 중복 활성 계획 방지

- **Given** TQQQ에 대한 활성 계획이 이미 존재하는 상태
- **When** TQQQ로 새 계획 생성을 시도한다
- **Then** 에러 메시지가 표시되고 계획이 생성되지 않는다

성공 기준:
- [ ] "해당 종목에 이미 진행 중인 계획이 있습니다" 에러 메시지가 표시된다
- [ ] 기존 활성 계획이 완료 상태가 되면 동일 종목으로 새 계획 생성이 가능하다

### 10. 계획 완료

- **Given** TQQQ 계획의 진행 상황이 "N-1 / N일" (N = 설정한 분할 횟수)인 상태
- **When** 마지막(N번째) 매수를 기록한다
- **Then** 계획이 완료 상태로 전환되고 활성 목록에서 완료 목록으로 이동한다

성공 기준:
- [ ] 진행 상황이 "N / N일 완료"로 표시된다
- [ ] 완료된 계획은 "진행 중" 목록에서 제거되고 "완료된 계획" 이력에 나타난다
- [ ] 완료 후 동일 종목(TQQQ)으로 새 계획 생성이 가능하다
- [ ] 완료된 계획에서 최종 평균단가와 최종 목표 매도가가 표시된다

### 11. 매도 — N/2 회차 이하 전량 매도

- **Given** TQQQ 계획이 15/40일 진행 중 (≤ 20회차)이고 현재가 ≥ 평균단가 × 1.10인 상태
- **When** 홈 배너의 [매도하기]를 탭하고 매도 확인 화면에서 단가를 확인한 뒤 [전량 매도]를 탭한다
- **Then** 전체 보유 수량이 매도되고 실현손익이 표시된다

성공 기준:
- [ ] 매도 확인 화면에 현재가(자동 채움, 수정 가능), 보유 수량 전체, 예상 실현손익이 표시된다
- [ ] [전량 매도] 탭 후 TQQQ 보유 수량이 0이 된다
- [ ] 실현손익 = (매도 단가 - 계획 평균단가) × 매도 수량 이 표시된다
- [ ] 전량 매도 후 TQQQ가 "보유 중" 목록에서 제거된다
- [ ] 계획은 매수 횟수와 무관하게 매도 완료 상태로 이력에 남는다

### 12. 매도 — N/2 회차 초과 1차 매도 (+5%, 50%)

- **Given** TQQQ 계획이 25/40일 진행 중 (> 20회차)이고 1차 매도 미완료이며 현재가 ≥ 평균단가 × 1.05인 상태
- **When** 홈 배너의 [매도하기]를 탭하고 매도 확인 화면에서 [1차 매도]를 탭한다
- **Then** 보유 수량의 50%가 매도되고 계획이 계속 활성 상태를 유지한다

성공 기준:
- [ ] 매도 확인 화면에 현재가(자동 채움, 수정 가능), 매도 수량(보유 수량 × 50%, 수정 가능), 예상 실현손익이 표시된다
- [ ] [1차 매도] 탭 후 TQQQ 보유 수량이 기존의 약 50%로 감소한다
- [ ] 1차 매도 후 계획은 완료되지 않고 활성 상태를 유지한다 (남은 일수 매수 계속 가능)
- [ ] 계획 상세에서 "1차 매도 완료" 상태가 표시된다
- [ ] 1차 매도 완료 후 홈 배너에서 "+5% 배너"가 사라진다

### 13. 매도 — N/2 회차 초과 2차 매도 (+10%, 나머지 전량)

- **Given** TQQQ 계획이 1차 매도 완료 상태이고 현재가 ≥ 평균단가 × 1.10인 상태
- **When** 홈 배너의 [매도하기]를 탭하고 매도 확인 화면에서 [2차 매도]를 탭한다
- **Then** 남은 보유 수량 전체가 매도되고 계획이 종료된다

성공 기준:
- [ ] 매도 확인 화면에 현재가(자동 채움, 수정 가능), 남은 보유 수량 전체, 예상 실현손익이 표시된다
- [ ] [2차 매도] 탭 후 TQQQ 보유 수량이 0이 된다
- [ ] 2차 매도 후 TQQQ가 "보유 중" 목록에서 제거된다
- [ ] 1차·2차 실현손익이 각각 계획 이력에 기록된다
- [ ] 보유 수량을 초과하는 수량 입력 시 "보유 수량을 초과합니다" 에러가 표시된다

### 14. 포트폴리오 요약 (홈)

- **Given** 여러 활성 계획이 진행 중이고 현재가가 조회된 상태
- **When** 홈 화면을 확인한다
- **Then** 전체 포트폴리오 합산 수치가 요약 섹션에 표시된다

성공 기준:
- [ ] 총 투자금(모든 계획의 누적 매수금액 합계)이 표시된다
- [ ] 총 평가금액(보유 수량 × 현재가 합계)이 표시된다
- [ ] 총 미실현손익(평가금액 - 투자금)과 수익률(%)이 표시된다
- [ ] 현재가 미조회 종목이 있을 경우 해당 종목을 제외한 나머지로 요약이 계산된다

### 15. 계획별 진행도 표시

- **Given** TQQQ 계획이 15/40일 진행 중, 평균단가 $48.50, 목표 수익률 10%인 상태
- **When** 홈 또는 계획 목록 화면을 확인한다
- **Then** 진행도, 목표가, 현재 단계에 맞는 매도 기준이 표시된다

성공 기준:
- [ ] 진행도 바(bar)와 함께 "15 / 40일"이 표시된다
- [ ] 잔여 투자금(총 투자금 - 지금까지 사용한 금액)이 표시된다
- [ ] 진행 회차 ≤ N/2인 계획: 목표 매도가($53.35 = $48.50 × 1.10)와 현재가 대비 목표가까지 남은 %(예: "목표까지 +4.2%")가 표시된다
- [ ] 진행 회차 > N/2이고 1차 매도 미완료인 계획: 1차 매도가($50.93 = $48.50 × 1.05)와 2차 매도가($53.35 = $48.50 × 1.10)가 함께 표시된다
- [ ] 1차 매도 완료 후: "1차 매도 완료" 상태와 2차 매도가만 표시된다

### 16. 티커 등록

- **Given** 티커 목록이 비어 있는 상태
- **When** "TQQQ"를 입력하고 추가 버튼을 탭한다
- **Then** 티커 목록에 "TQQQ"가 나타나고 계획 생성 시 선택 가능한 상태가 된다

성공 기준:
- [ ] 추가 후 티커 목록에 "TQQQ" 행이 나타난다
- [ ] 이미 등록된 티커를 재입력하면 "이미 등록된 종목입니다" 에러 메시지가 표시된다
- [ ] 빈 값 입력 시 추가 버튼이 비활성화된다

### 17. 티커 삭제

- **Given** "TQQQ" 티커가 등록된 상태
- **When** TQQQ 삭제를 시도한다
- **Then** 활성 계획 또는 매수 기록 유무에 따라 삭제 가능 여부가 결정된다

성공 기준:
- [ ] TQQQ에 활성 계획이나 매수 기록이 있을 때 삭제 시도 → "거래 또는 계획이 있는 종목은 삭제할 수 없습니다" 에러가 표시된다
- [ ] 아무 기록도 없는 티커 삭제 시 → 확인 다이얼로그 후 목록에서 제거된다

## 불변 규칙

**즉시 반영**: 매수·매도 기록 후 평균단가·목표 매도가·진행도·포트폴리오 요약이 페이지 새로고침 없이 즉시 업데이트된다.

**거래일 기준 카운트**: 계획 진행 카운트는 실제로 매수 기록이 저장된 날(거래일)만 포함한다. 주말·미국 공휴일·의도적으로 건너뛴 날은 카운트되지 않는다.

**계획 평균단가 독립**: 계획의 평균단가는 해당 계획에 속한 매수 기록만 집계한다. 다른 계획의 동일 종목 거래는 포함하지 않는다.

**수수료 분리**: 수수료는 평균단가 계산에 포함하지 않으며, 계획 상세에서 총 수수료를 별도 항목으로 표시한다.

**현재가 실패 시 기존값 유지**: API 조회 실패 시 이전 캐시 가격과 조회 시각을 그대로 표시한다.

**목표 매도가 단일 표시**: 매수 기록마다 목표 매도가(= 계획 평균단가 × 목표 수익률)가 즉시 갱신되며, 항상 가장 최신 값 하나만 표시된다.

**매수는 계획에 귀속**: 모든 매수 기록은 반드시 활성 계획에 속한다. 계획 없는 독립 매수 거래는 존재하지 않는다.

**진행 회차별 매도 기준**: 분할 횟수 N에 대해 진행 회차가 N/2 이하이면 평균단가 × (1 + 목표 수익률) 도달 시 전량 매도가 적용된다. 진행 회차가 N/2 초과이면 평균단가 × 1.05 도달 시 1차 50% 매도, 이후 평균단가 × (1 + 목표 수익률) 도달 시 2차 나머지 전량 매도가 적용된다. 1차 매도가 완료된 후에도 남은 일수의 매수는 계속 진행할 수 있다.
